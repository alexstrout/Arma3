Enables configuration of all installed "foxMod" module settings via text config file.

This is only required if you wish to configure any of the options provided in [url=http://steamcommunity.com/sharedfiles/filedetails/?id=1052311807]other foxMod modules[/url].

[h1]Usage[/h1]
[list][*]Navigate to the mod directory.
(Launcher -> Mods -> Right-Click "foxMod Configuration" -> Open folder)
[*]Copy the "userconfig" folder to your Arma 3 install directory.
e.g. "C:\Program Files (x86)\Steam\steamapps\common\Arma 3"
[*]Edit "<Arma 3 install dir>\userconfig\foxConfig\settings.hpp" in any text editor as desired.
More information on each option can be found within the file itself.[/list]

[h1]Server Information[/h1]
Public server key is included in the "keys" folder.
This mod is client-side, and has no effect when run directly on a dedicated server.
